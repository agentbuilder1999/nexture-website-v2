# Capsule Endoscopy — 基础知识

## 什么是胶囊内镜
患者吞服一粒胶囊大小的摄像机，在消化道内拍摄 8-10 小时，产生约 5-8 万张图片。
主要用于检查小肠（传统内镜无法到达）。

## 主要检查目标
- 血管畸形/扩张（Angiectasia）— 常见出血原因
- 消化道出血（急性/慢性）
- 克罗恩病
- 息肉、肿瘤
- 溃疡、糜烂

## 医生工作流程
1. 患者完成检查，图片上传系统
2. 医生逐帧审阅（传统方式需 2-4 小时）
3. 标记可疑帧，写报告
4. 报告发给患者和转诊医生

## TheraSeus 介入点
步骤 2 前：AI 预处理，过滤重复/无意义帧，标注可疑帧
医生只需审阅 AI 标注的关键帧 + 抽样验证 = 节省 60%+ 时间

## 关键指标
- 灵敏度（Sensitivity）：有临床意义的帧被正确保留的比例（越高越好，≥95%）
- 特异度（Specificity）：无意义帧被正确过滤的比例
- 假阴性率（FNR）：有意义帧被错误过滤的比例（必须极低，<1%）
